## 🛠️ Installation

Depending on your operating system and launcher, use the recommended methods below:

<details>
<summary>🪟 Windows</summary>

Choose your Launcher:

<details>
<summary>Standard Minecraft Launcher</summary>

1. Press **Win + R**, type `%appdata%\.minecraft\resourcepacks`, and hit **Enter**.
2. Drag and drop the downloaded `.zip` file into this folder.
3. In-game, go to **Options > Resource Packs** and move **Better-GUI** to the right side.
</details>

<details>
<summary>Prism / MultiMC / CurseForge</summary>

1. Open your Launcher.
2. **Right-click** your Minecraft instance and select **"Open Folder"** (or "Open Minecraft Folder").
3. Navigate into the `resourcepacks` folder.
4. Drop the `.zip` file here.
</details>
</details>

<details>
<summary>🍎 MacOS</summary>

Choose your Launcher:

<details>
<summary>Standard Minecraft Launcher</summary>

1. In Finder, select **Go > Go to Folder...** from the top menu bar.
2. Paste: `~/Library/Application Support/minecraft/resourcepacks`
3. Drop the `.zip` file here and activate it in-game.
</details>

<details>
<summary>Prism / MultiMC / CurseForge</summary>

1. Open the Launcher.
2. **Right-click** your instance and select **"Open Minecraft Folder"**.
3. Open the `resourcepacks` directory and paste the `.zip` file.
4. *Manual path (Prism):* `~/Library/Application Support/PrismLauncher/instances/`
</details>
</details>

<details>
<summary>🐧 Linux</summary>

Choose your Launcher:

<details>
<summary>Standard Minecraft Launcher</summary>

1. Navigate to `~/.minecraft/resourcepacks`.
2. Move the `.zip` file into this directory.
3. Launch the game and enable it under **Resource Packs**.
</details>

<details>
<summary>Prism / MultiMC (Recommended)</summary>

1. Right-click your instance in the launcher and click **"Open Minecraft Folder"**.
2. Navigate to the `resourcepacks` folder and paste the `.zip`.
3. **Note for Flatpak users:** If you cannot find the folder manually, the path is usually:  
   `~/.var/app/org.prismlauncher.PrismLauncher/data/PrismLauncher/instances/`
</details>

<details>
<summary>CurseForge (Linux Beta)</summary>

1. Open the CurseForge App.
2. Click the **three dots** next to your instance and select **"Open Folder"**.
3. Place the `.zip` in the `resourcepacks` subfolder.
</details>
</details>

> [!IMPORTANT]  
> **Installation Note:** If you have other versions of the Better GUI pack already installed, it is recommended that you remove the earlier installations to prevent the resource pack menu from becoming cluttered.
